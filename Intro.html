Video Files and Example

However, in the real world, Video support is more complicated.

  • Older Browser Support. For older browsers, you should really have Adobe Flash fallback support but that isn't covered in this class.
  • Video Resolution. How do you make sure that viewers who have slower video speed can still watch the content? Ideally, you would have a one, or two, or both of these solutions:
    • You would have something that automatically detects the user's internet speed, and adjust the video resolution accordingly.
    • Have some level of resolution picker so that the user can select the file size they would like.
  • Custom Branding What if you want to control how the pause or play buttons look and function? You would have to use JavaScript in order to override the HTML controler. Again, not covered in this class.

All of these issues have the same issue and solution: They are outside the scope of basic HTML programming Without JS, you would be better off
using a third party video service.

Special Characters

In HTML you don't want to print symbols to the screen if you want them to display as content. I actually just ran into this when writting up the description of different lists above. All of the symbols are things you would use in the code, and it confuses the compiler and is bad practice. Instead, use the following codes:

  • Right single quote: ’
  • Left single quote: ‘
  • Right doble quote: ”
  • Ampersands: &
  • Greater than: >
  • Less than: <
  • Copyright: ©
  • Registered symbol: ®
  • Trademark symbol: ™
